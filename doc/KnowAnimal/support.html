应用简介

KnowAnimal 是一款专为 2-8 岁儿童 设计的动物认知应用，包含：

  • 200 种动物：覆盖农场、森林、海洋、飞鸟、昆虫、爬行、极地 7 大类别

常见问题 FAQ

Q1：应用需要联网吗？

不需要。所有动物图片、叫声、文字内容均已打包在应用内，离线即可正常使用。仅在「恢复购买」「管理订阅」「查看 App Store 商品价格」时需要短暂联网。

Q2：会收集我或孩子的个人信息吗？

不会。我们不收集任何个人信息、不接入任何第三方分析或广告 SDK、不要求注册登录。详见 隐私政策。

Q3：应用支持哪些语言？

当前支持简体中文（zh-Hans）与美式英语（en-US）。可在应用内「个人中心 → 设置 → 显示英文」切换。

Q4：应用是否包含广告？

不包含任何广告。免费用户也不会看到任何广告。

Q5：付费后可以在多台设备共享吗？

可以。终身买断与订阅均支持苹果「家庭共享」，最多 6 名家庭成员共享。具体设置参见苹果官方说明：https://support.apple.com/zh-cn/HT201079

Q6：换手机后付费记录会丢失吗？

不会。只要新手机登录的是同一个 Apple ID，进入应用后点击「恢复购买」即可还原所有已购权益。

故障排查

动物叫声播放不出来

  1. 检查 iPhone 静音键是否打开
  2. 检查媒体音量是否调到最低
  3. 进入应用「个人中心 → 设置 → 音效」开关是否开启
  4. 重启应用

应用闪退

  1. 重启 iPhone
  2. 删除应用后重新安装（注意：删除后本地收藏、学习进度会丢失，但已购权益登录同一 Apple ID 后可通过「恢复购买」恢复）
  3. 检查 iOS 系统是否为最新版本

Paywall 价格显示异常 / 无法购买

  1. 检查 iPhone「设置 → 顶部 Apple ID → 媒体与购买项目」中账号区域是否为中国大陆
  2. 检查 Apple ID 是否绑定有效支付方式
  3. 检查是否开启了「屏幕使用时间」中的内购限制
  4. 联网后重启应用重试

「恢复购买」无效

  1. 确认当前 Apple ID 与原购买时使用的是同一账号
  2. 联网状态下重新点击「恢复购买」按钮
  3. 仍无效请联系客服并提供原购买凭证

取消订阅

  1. 打开 iPhone「设置」
  2. 点击顶部 Apple ID 头像
  3. 选择「订阅」
  4. 找到「KnowAnimal Premium」
  5. 点击「取消订阅」

取消后，订阅在当前付费周期结束前仍可正常使用。

家长须知

KnowAnimal 严格遵循儿童隐私保护原则：

  • 零数据收集：不收集姓名、邮箱、位置、设备标识、使用行为等任何信息
  • 零第三方追踪：不接入广告、分析、社交 SDK
  • 零外链跳转：应用内无浏览器、无外部链接
  • 付费由家长门把控：所有付费入口需通过家长门（家长完成简单算术题）方可进入
  • 应用全程符合 COPPA（美国儿童在线隐私保护法）与 个人信息保护法（中国）要求

详见 隐私政策。
