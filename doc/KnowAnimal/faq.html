KnowAnimal 常见问题 · FAQ

最近更新：2026 年 4 月 29 日

关于应用

Q：KnowAnimal 适合多大孩子？

A：2-8 岁。低龄段（2-4 岁）侧重图片+叫声认知；中高龄段（5-8 岁）侧重知识问答与记忆游戏。

Q：包含哪些动物？

A：200 种，分 7 大类别：农场、森林、海洋、飞鸟、昆虫、爬行、极地。每种动物配高清照片与真实叫声。

Q：必须联网吗？

A：不需要。所有内容已打包至应用内，离线即可使用。仅「恢复购买」「加载商品价格」需要短暂联网。

Q：有广告吗？

A：没有。免费版与付费版都不会出现任何广告。

Q：会收集隐私信息吗？

A：完全不会。零数据收集是 KnowAnimal 的核心承诺。详见 隐私政策。

关于付费

Q：免费版能用什么？

A：免费版可体验首批动物与基础功能。完整 200 种动物与全部游戏需购买会员或终身买断解锁。

Q：三种付费方式怎么选？

A：

  • 想长期用：选 ¥28 终身买断，最划算
  • 不确定是否长期用：选 ¥18 年度订阅（含 3 天免费试用），先试再买
  • 临时使用：选 ¥3 月度订阅

Q：订阅会自动续费吗？怎么取消？

A：会。在 iPhone「设置 → 顶部 Apple ID → 订阅 → KnowAnimal Premium → 取消订阅」可随时取消。取消后当前周期结束前仍可使用。

Q：3 天免费试用怎么用？

A：选择年度订阅时，自动开启 3 天免费试用。试用期内随时取消不扣费；超过试用期未取消，按 ¥18/年扣费。每个 Apple ID 仅一次试用机会。

Q：换手机后付费会丢吗？

A：不会。新手机登录同一 Apple ID，进入应用点「恢复购买」即可。

Q：能家庭共享吗？

A：可以。所有付费商品均支持苹果家庭共享，最多 6 名成员共享。

Q：怎么申请退款？

A：苹果订阅退款由苹果统一处理，前往 https://reportaproblem.apple.com 提交申请。

故障排查

Q：应用闪退怎么办？

A：

  1. 重启 iPhone
  2. 删除应用重新安装（本地收藏会丢失，但已购权益登录原 Apple ID 后可「恢复购买」）
  3. 检查 iOS 是否为最新版本

Q：动物叫声播放不出来？

A：

  1. 检查 iPhone 静音键
  2. 检查媒体音量
  3. 应用内「个人中心 → 设置 → 音效」是否开启
  4. 重启应用

Q：购买时一直转圈？

A：检查网络连接，并在「设置 → Apple ID → 媒体与购买项目」确认账号有效。仍异常请联系客服。

Q：买了为什么提示未付费？

A：点击应用内「恢复购买」按钮即可。仍异常请联系客服并提供原购买凭证（App Store 邮件收据）。
