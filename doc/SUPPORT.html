About zz markdown

zz markdown is a lightweight Markdown editor for macOS with real-time preview, syntax highlighting, and multiple themes.

System Requirements

  • macOS 14.0 (Sonoma) or later
  • Apple Silicon or Intel Mac

Frequently Asked Questions

How do I open a Markdown file?

Use File → Open or drag and drop a .md file onto the app icon.

What file formats are supported?

.md, .markdown, .mdown, and .mkd files are supported.

How do I switch between themes?

Click the theme icon (sun/moon) in the toolbar to switch between Light, Dark, and GitHub themes.

How do I export to PDF?

Click the export icon in the toolbar and select “Export as PDF”.

Does the app collect any data?

No. zz markdown operates entirely offline and does not collect any personal data. See our Privacy Policy for details.
