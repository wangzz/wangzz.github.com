# KnowAnimal · 认识动物

> 最安全的儿童动物认知伙伴 · 完全离线 · 零数据收集

KnowAnimal 是一款为 2-8 岁儿童打造的动物认知应用，包含 200 种动物、7 大类别、4 种互动游戏，全部内容已打包在应用内，**无需联网、无需注册、不收集任何用户数据**。
