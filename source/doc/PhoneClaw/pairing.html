配对你的 OpenClaw 主机

在运行 OpenClaw 的电脑上安装 PhoneClaw Skill，然后生成配对码并连接到 PhoneClaw App

手动安装 PhoneClaw Skill
在你的电脑上打开终端，按以下步骤安装并配对。
  • 打开终端，运行 npm install -g phoneclaw-connector@latest 全局安装 PhoneClaw Skill
  • 确认电脑已安装 OpenClaw（Skill 会自动从 ~/.openclaw/openclaw.json 读取网关配置，无需手动填写任何文件）
  • 运行 phoneclaw pair 启动配对，终端会显示 6 位配对码
  • 在 PhoneClaw App 中输入配对码完成配对
  • 配对成功后，运行 phoneclaw start 让 Skill 保持后台运行，之后即可在手机端与电脑端交互
