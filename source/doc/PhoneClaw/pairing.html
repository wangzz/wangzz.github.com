配对你的 OpenClaw 主机

在运行 OpenClaw 的电脑上安装 PhoneClaw Skill，然后生成配对码并连接到 PhoneClaw App

在电脑上打开配对网页
在运行 Agent 的电脑上打开这个网页，复制安装/配对 prompt，再生成配对码。
  • 在电脑浏览器中打开配对网页
  • 复制下方的安装指令，发送给 Agent
  • 等待 Agent 安装完成后，让它生成配对码
  • 在 PhoneClaw App 中输入配对码完成连接
    • 打开配对网站
    让 Agent 帮忙安装 PhoneClaw Skill 推荐
    复制下面这段话，直接发给 Agent，让它先安装 PhoneClaw Skill，再把这台 OpenClaw 主机配对到 PhoneClaw App。
    请先从这个仓库安装 phone-claw-skill，并严格按下面要求执行：https://github.com/wangzz/PhoneClaw/tree/main/phone-claw-skill 1. 先直接运行 `npm install -g pm2`，确保本机有 pm2（用于后台运行）。 2. 将 phone-claw-skill 文件夹复制到本机任意位置，例如 `~/phone-claw-skill`。 3. 进入目录后执行 `npm install && npm run build` 安装依赖并编译。 4. 使用 `pm2 start dist/index.js --name phoneclaw-skill` 启动 Skill，让它连接到中继服务器。 5. 启动成功后，执行 `pm2 logs phoneclaw-skill --lines 20` 查看日志，确认看到 "Relay connected" 和 "Initialization complete"。 6. 然后执行配对命令生成配对码（Skill 会在终端显示 6 位配对码）。 7. 严格按 skill 的步骤执行，不要跳过检查。 8. 如果成功，请直接告诉我配对码是多少，以及下一步应该执行的命令。 9. 如果失败，请直接告诉我卡在哪一步，以及下一步应该执行的命令。 这台主机应按 OpenClaw 配对。生成配对码时请使用 skill 提供的配对功能。
